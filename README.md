<p align="center">
  <strong>Project Features</strong>
</p>

- Feature 1: I have used jwt based authentication for securing user data.
- Feature 2: Here user can add a item to their cart and edit the cart if they want.
- Feature 3: User can Pay here through paypal for their items.
- Feature 4: User can review a product based on their experience.
- Feature 5: On admin dashboard ,admin can approve or cancel a order.
